# 🤖 RAG Knowledge Assistant (No Hallucination AI)

An intelligent AI system that answers questions strictly from your documents — ensuring **accuracy, reliability, and zero hallucination**.

---

## 🚀 Overview

Most AI systems are powerful but unreliable — they **guess when unsure**.

This project solves that problem by implementing a **Retrieval-Augmented Generation (RAG)** pipeline that:

- 📄 Understands uploaded documents (PDF/Text)
- 🔍 Retrieves only relevant information
- 🧠 Generates answers strictly from that data
- ❌ Refuses to guess if answer is not found

👉 Result: **Trustworthy AI for real-world use cases**

---

## ✨ Features

- 📚 **Document Upload (PDF/Text)**
- 🔍 **Semantic Search using Embeddings**
- ⚡ **Fast Retrieval with FAISS**
- 🧠 **Context-Aware AI Responses**
- 🚫 **No Hallucination System** (Strict grounding)
- 🔄 **Query Rewriting** for better search accuracy
- 🎨 **Modern UI (Next.js + Glassmorphism)**
